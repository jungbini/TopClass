  Un-break MultiArtifactCache and prevent silencing of errors.  Summary:MultiArtifactCache would cause an unconditional failure if >1 cache supported stores. It would also cause errors if an un-borrowable path was passed in to the store method. Finally the BuildInfoRecorder would silently ignore any of these errors. This fixes all of the above and adds an integration case for the 'full' cache stack (dir+http+2-level).
