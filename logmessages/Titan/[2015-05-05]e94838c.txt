If a test ran in multiple executions, then each execution of said test after the first would overwrite the result files left by the previous execution of the same test.  Now the result files are suffixed by the short strings in the parentheticals above, which should prevent result overwrites.  This commit has no effect on Maven's ability to detect failures.
