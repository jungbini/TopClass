  Fix flakiness due to the MemoryStatus output  The test `MemoryStatusUpdateIntegrationTest.can register a listener for JVM and OS memory status update events` register listener for JVM memory and OS memory on `MemoryManager`. This service is owned by `GlobalScopeServices`. In embedded test scenario, the listener stays registered after the test generating additional output in the following tests. This fix ensure the listener are removed from the service before the test ends.
